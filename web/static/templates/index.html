Detalhes da busca

Código de 2 letras — pt para português, en para inglês.

📍 Localização (busca por raio)

Preencha latitude/longitude para centralizar a busca num ponto e use o raio (em Modo Rápido) para limitar por distância.

⚙️ Opções avançadas

Uma única requisição por palavra-chave (rápido). Exige latitude/longitude preenchidos.

Ex: 10m, 1h, 30m. Mínimo de 3 minutos.
